Product Requirements Document (PRD)
Project: Chinese Character Block — Landing Page

Owner: Lin Li
Platform: Static website (HTML, CSS, minimal JS)
Deployment: Hosted on Vercel
Development Stack: Flake + Python for build scripting, HTML5, CSS3
Target Systems: Windows & macOS browsers, responsive for mobile

1. Project Goals

Create a simple, visually appealing, and responsive landing page for the Chinese Character Block game that includes:

A game demo video showing gameplay.

A download button for the .exe file (in /public).

A short game description.

A PayPal donation button (email hidden).

Clean, modern, responsive design (like webpage_example.png).

Deployed on Vercel as a static site.

2. Visual Design Requirements

Reference: webpage_example.png

Key Design Elements:

Centered layout (vertical and horizontal).

Soft educational background: sky blue gradient or soft green ground.

Rounded buttons with hover animation.

Modern sans-serif font: Poppins, Nunito Sans, or Inter.

Color scheme:

Background: #C8EBF9

Primary button: #F28C38

Button hover: #FFD89C

Text: #333333

Accent (borders/shadows): rgba(0,0,0,0.1)

3. Responsiveness
Device	Behavior
Desktop	Video centered at 720px width, text centered below.
Tablet	Video scaled to 80% width, two buttons per row.
Mobile	Video 100% width, single-column layout, stacked buttons.
